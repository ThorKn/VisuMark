getOrdersContainingZips(zips, sort=['c'])[source]¶

Get orders, optionally limiting them by a list of zip codes. The elements in zips are regarded as the first letters a zip code must contain. sort can contain ‘a’ for articles, ‘s’ for shops and ‘z’ for zips and ‘d’ for date in any order. The order of the elements in sort defines the sort order

getOrdersMultiFilter(articles: List[Union[int, str]] = [], shops: List[Union[int, str]] = None, customers: List[Union[int, str]] = None, sort: List[str] = [])[source]¶

Get orders, limit them by multiple filters for articles, shops and customers. sort can contain ‘a’ for articles, ‘s’ for shops and ‘c’ for customers and ‘d’ for date in any order. The order of the elements in sort defines the sort order
